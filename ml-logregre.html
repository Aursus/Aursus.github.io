Logistic Regression

Utilization: Curve fitting, classification into two classes, typically deals with data points.

Applicability: Can be applied universally. If data separation can be achieved with a single line, logistic regression can be used.

Input Space

X=Rd\mathcal{X}=\mathbb{R}^d X=Rd

Label Space/Output Space (y ranges from 0 to 1)

Y=[0,1]\mathcal{Y}=[0,1] Y=[0,1]

Hypothesis Class FFF

F:={x↦sigmoid(w⊤x+b) ∣ w∈Rd,b∈R},where sigmoid(a)=11+exp⁡(−a)\mathcal{F}:=\{x \mapsto \text{sigmoid}(w^\top x+b)\ |\ w\in\mathbb{R}^d, b\in\mathbb{R}\}, \text{where } \text{sigmoid}(a)=\frac{1}{1+\exp(-a)} F:={x↦sigmoid(w⊤x+b) ∣ w∈Rd,b∈R},where sigmoid(a)=1+exp(−a)1​

Note: ω\omegaω: weight vector; bbb: bias

Decision Boundary

The result of P(y=1∣x)P(y=−1∣x)=1+exp⁡(w⊤x)1+exp⁡(−w⊤x)=exp⁡(w⊤x)\dfrac{P(y=1|x)}{P(y=-1|x)}=\dfrac{1+\exp(w^\top x)}{1+\exp(-w^\top x)}=\exp(w^\top x)P(y=−1∣x)P(y=1∣x)​=1+exp(−w⊤x)1+exp(w⊤x)​=exp(w⊤x) indicates:

  • If w⊤x=0w^\top x=0w⊤x=0, then P(y=1∣x)=P(y=−1∣x)P(y=1|x)=P(y=-1|x)P(y=1∣x)=P(y=−1∣x).
  • If w⊤x>0w^\top x>0w⊤x>0, then P(y=1∣x)>P(y=−1∣x)P(y=1|x)>P(y=-1|x)P(y=1∣x)>P(y=−1∣x), indicating a higher probability for label=1.
  • If w⊤x<0w^\top x<0w⊤x<0, then P(y=1∣x)<P(y=−1∣x)P(y=1|x)<P(y=-1|x)P(y=1∣x)<P(y=−1∣x), indicating a higher probability for label=-1.
  • If w⊤x=0w^\top x=0w⊤x=0, then P(y=1∣x)P(y=−1∣x)=exp⁡(w⊤x)\dfrac{P(y=1|x)}{P(y=-1|x)}=\exp(w^\top x)P(y=−1∣x)P(y=1∣x)​=exp(w⊤x), precisely at the decision boundary.
