Binary Classification

Assumptions

  1. Linearly separable data.
  2. Binary classification, meaning the data can be divided into two classes.

Objective

To categorize the data into two classes based on labels.

Set up

Input space

X=Rd\mathcal{X} = \mathbb{R}^d X=Rd

Label space / Output space

Y={−1,1}\mathcal{Y} = \{-1, 1\} Y={−1,1}

Classifier | Hypothesis class F\mathcal{F}F

F:={x↦sign(w⊤x+b) ∣ w∈Rd,b∈R}\mathcal{F} := \{x \mapsto \text{sign}(\mathbf{w}^\top \mathbf{x} + \mathbf{b})\ |\ \mathbf{w}\in\mathbb{R}^d,\mathbf{b}\in\mathbb{R}\} F:={x↦sign(w⊤x+b) ∣ w∈Rd,b∈R}

Where ω\omegaω is the weight vector; bbb is the bias, and sign is the sign function defined as:

sign(a)={1if a≥0−1otherwise\text{sign}(a)=\left\{ \begin{aligned} &1 \qquad &&\text{if}\ a\geq0\\ &-1 \qquad &&\text{otherwise} \end{aligned} \right. sign(a)={​1−1​​if a≥0otherwise​

To simplify subsequent calculations, we aim to have the classifier containing only www without bbb. For this purpose, we add a dimension and treat bbb in the second dimension of www as a constant, as follows:

x↦[x1]w↦[wb]x \mapsto \left[ \begin{matrix}x\\1 \end{matrix}\right] \quad w \mapsto \left[ \begin{matrix}w\\b \end{matrix}\right] x↦[x1​]w↦[wb​]

The original classifier simplifies to

F:={x↦sign(w⊤x) }\mathcal{F} := \{x \mapsto \text{sign}(\mathbf{w}^\top \mathbf{x})\ \} F:={x↦sign(w⊤x) }

Empirical Risk Minimizer

We use the following loss function to find ERM:

ℓ(f(x),y)={0if f(x)=y1otherwise\ell(f(x),y)=\left\{ \begin{aligned} &0 \qquad &&\text{if}\ f(x)=y\\ &1 \qquad &&\text{otherwise} \end{aligned} \right. ℓ(f(x),y)={​01​​if f(x)=yotherwise​

This expression is also known as 0-1 loss, where 1 represents prediction error, and 0 represents correct prediction. This expression can also be represented using the indicator function I\mathbb{I}I (Indicator function) as:

I[f(x)≠y]={1if f(x)≠y0if f(x)=y\mathbb{I}[f(x)\neq y] = \left\{ \begin{aligned} &1 \qquad &&\text{if}\ f(x)\neq y\\ &0 \qquad &&\text{if}\ f(x)= y \end{aligned} \right. I[f(x)​=y]={​10​​if f(x)​=yif f(x)=y​

In the case of linearly separable data, w^\hat ww^ is referred to as w∗w^*w∗. At this point, w∗w^*w∗ satisfies yi=sign(ω∗⊤xi)y_i=sign(\omega_*^\top x_i)yi​=sign(ω∗⊤​xi​) for any point, and ω∗⊤xi\omega_*^\top x_iω∗⊤​xi​ serves as the boundary between the two categories.

Perceptron Algorithm

Objective

Update www until all data points are correctly classified.

Applicability

  1. Data points must be completely linearly separable.
  2. Once data like XOR or non-linearly separable data occurs, the perceptron cannot be used.
    1. XOR and non-linearly separable data can be segmented using kernel SVM.

Algorithm

Update Rule

If misclassified, i.e., sign(w⊤xi)≠yi\text{sign}(w^\top x_i)\neq y_isign(w⊤xi​)​=yi​ or sign(yiw⊤xi<0)\text{sign}(y_iw^\top x_i<0)sign(yi​w⊤xi​<0), update according to rule wt+1=wt+yixiw_{t+1}=w_t+y_ix_iwt+1​=wt​+yi​xi​.

If yi=1y_i=1yi​=1, then wt+1⊤xi=wt⊤xi+∥xi∥2>wt⊤xiw_{t+1}^\top x_i = w_t^\top x_i +\|x_i\|^2 > w_t^\top x_iwt+1⊤​xi​=wt⊤​xi​+∥xi​∥2>wt⊤​xi​. w updates towards the positive direction.

If yi=−1y_i=-1yi​=−1, then wt+1⊤xi=wt⊤xi−∥xi∥2<wt⊤xiw_{t+1}^\top x_i = w_t^\top x_i -\|x_i\|^2 < w_t^\top x_iwt+1⊤​xi​=wt⊤​xi​−∥xi​∥2<wt⊤​xi​. w updates towards the negative direction.
