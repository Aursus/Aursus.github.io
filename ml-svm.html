Linear SVM

Set up

Given n data points (x1,y1),...,(xn,yn)(x_1, y_1), ..., (x_n, y_n)(x1​,y1​),...,(xn​,yn​), where yiy_iyi​ is 1 or -1, indicating which class xix_ixi​ belongs to. Each xix_ixi​ is a p-dimensional real vector.

The goal of SVM is to find a maximum-margin hyperplane that separates all points xix_ixi​ based on yi=1y_i=1yi​=1 and yi=−1y_i=-1yi​=−1, ensuring the maximum distance from the hyperplane to the nearest points of both groups.

SVM Objective: Find a hyperplane that maximizes the margin.

Hyperplane Definition:

ω⊤xi+b=0\omega^\top x_i + b = 0 ω⊤xi​+b=0

Margin Definition: The distance of the nearest point to the hyperplane.

γ(w,b)=min⁡i∈[m]∣w⊤xi+b∣∥w∥2\gamma(w,b) = \min_{i\in[m]}\frac{|w^\top x_i + b|}{\|w\|_2} γ(w,b)=i∈[m]min​∥w∥2​∣w⊤xi​+b∣​

Hard Margin SVMs

Assumption: Data can be completely separated by hyperplane ω∗⊤x+b∗=0\omega_*^\top x + b_* = 0ω∗⊤​x+b∗​=0.

Objective: Find a classifier that obtains the maximum margin.

Approach 1: Geometric interpretation (wiki approach)

Firstly, two hyperplanes are created for the two groups of xix_ixi​ based on yi=1y_i=1yi​=1 and yi=−1y_i=-1yi​=−1:

For yi=1y_i=1yi​=1:

Hyperplane 1: ω⊤x−b=1\omega^\top x - b = 1ω⊤x−b=1, includes all data points above this hyperplane.

For yi=−1y_i=-1yi​=−1:

Hyperplane 2: ω⊤x−b=−1\omega^\top x - b = -1ω⊤x−b=−1, includes all data points below this hyperplane.

These two hyperplanes are parallel, and the maximum-margin hyperplane ω⊤x−b=0\omega^\top x - b = 0ω⊤x−b=0 lies between them.

Finally, all data points satisfy the following:

For yi=1y_i=1yi​=1: ω⊤xi−b≥1\omega^\top x_i - b \geq 1ω⊤xi​−b≥1

For yi=−1y_i=-1yi​=−1: ω⊤xi−b≤−1\omega^\top x_i - b \leq -1ω⊤xi​−b≤−1

Combining both, we get yi(ω⊤xi−b)≥1y_i(\omega^\top x_i - b) \geq 1yi​(ω⊤xi​−b)≥1 for all 1≤i≤n1 \leq i \leq n1≤i≤n.

Non-Linear Kernel SVM

Concept

Purpose: Address non-linearly separable data that cannot be solved simply by adding slack, such as two circles.

Approach: To separate, we map the data to a higher dimension using a feature map ϕ\phiϕ, i.e., x↦ϕ(x)x\mapsto \phi(x)x↦ϕ(x).

Method: Replace dot product with a non-linear kernel.

Example

Linearizing Data through ϕ\phiϕ to obtain a new predictor function

Using feature map ϕ\phiϕ, map the training data from a non-linear input space to a linear feature space:

{(x1,y1),...,(xm,ym)}↦{(ϕ(x1),y1),...,(ϕ(xm),ym)}\{(x_1,y_1),...,(x_m,y_m)\} \mapsto \{(\phi(x_1),y_1),...,(\phi(x_m),y_m)\} {(x1​,y1​),...,(xm​,ym​)}↦{(ϕ(x1​),y1​),...,(ϕ(xm​),ym​)}

Thus, we obtain a linear predictor function:

ω⊤ϕ(x)+b\omega^\top \phi(x)+b ω⊤ϕ(x)+b

Where ϕ(x)\phi(x)ϕ(x) is as follows, and ddd represents the dimensionality of xxx:

ϕ(x)=[1,x1,...,xd,x12,x1x2,...,xd2]⊤\phi(x) = [1,x_1,...,x_d,x_1^2,x_1x_2,...,x_d^2]^\top ϕ(x)=[1,x1​,...,xd​,x12​,x1​x2​,...,xd2​]⊤

Solving the optimization function using Soft-SVM’s method

Since direct solving is too high-dimensional, we use the Soft-SVM method to solve:

Equivalent to solving the inner product ϕ(x)⊤ϕ(x′)\phi(x)^\top \phi(x')ϕ(x)⊤ϕ(x′)

We can simplify it to:

ϕ(x)⊤ϕ(x′)=1+x⊤x′+(x⊤x′)2\phi(x)^\top \phi(x') = 1+x^\top x'+(x^\top x')^2 ϕ(x)⊤ϕ(x′)=1+x⊤x′+(x⊤x′)2

Constructing Kernels

To simplify the solving process, let the kernel k∈Rk\in\mathbb{R}k∈R be defined as:

k(xi,xj)=<ϕ(xi),ϕ(xj)>=ϕ(xi)⋅ϕ(xj)k(x_i,x_j)=<\phi(x_i),\phi(x_j)>=\phi(x_i)\cdot \phi(x_j) k(xi​,xj​)=<ϕ(xi​),ϕ(xj​)>=ϕ(xi​)⋅ϕ(xj​)

For x1,x2,...,xmx_1,x_2,...,x_mx1​,x2​,...,xm​, the kernel matrix K∈Rm×mK\in \mathbb{R}^{m\times m}K∈Rm×m composed of kkk is:

Kij=k(xi,xj)=<ϕ(xi),ϕ(xj)>K_{ij}=k(x_i,x_j)=<\phi(x_i),\phi(x_j)> Kij​=k(xi​,xj​)=<ϕ(xi​),ϕ(xj​)>

Where K is a symmetric and positive semi-definite matrix, satisfying these three conditions: 1) K=K⊤K=K^\topK=K⊤, 2) x⊤Kx≥0x^\top K x\geq 0x⊤Kx≥0, 3) all eigenvalues are non-negative.
