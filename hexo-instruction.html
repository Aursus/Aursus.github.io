hexo - 基本操作

发表于2024-03-07|更新于2024-03-07

常见操作

上传博客

  1. 本目录下右键 Git Bash Here
  2. 输入hexo g
  3. 输入hexo d
  4. 就可以在网页上打开了

中英双语：

使用./deploy.sh d all来upload

本地生成博客内容：

  1. 本目录下右键 Git Bash Here
  2. 输入hexo s则编译打开文件
  3. ctrl+C 结束分享

教程：https://www.bilibili.com/video/BV1mU4y1j72n?p=5&vd_source=2029748cdca0ccea25511170cf7a00c3

常见指令

创建post

hexo new <post名称>

本机post预览

hexo s

上传github

hexo g

发布

hexo d

创建draft

hexo new draft <draft名称>

本机草稿预览

hexo S --draft

发布草稿

hexo P <filename>

发布页面

hexo new page <name>

进阶操作

文章置顶

在github base中输入npm install hexo-generator-topindex --save

在需要置顶的文章上top命令，如top：1，数字越大置顶越前

中英双语

参考：https://aursus.github.io/hexo-bilingual.html
